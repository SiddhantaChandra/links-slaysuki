# Links Slaysuki

This app is similar to Linktree, but offers much greater customization for your links and appearance.

Create your own personalized link page, folks!

## This project is built with:

- React
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
	```
	npm install
	```
or
    ```
    bun install
    ```
2. Start the development server:
	```
	npm run dev
	```
or 
    ```
    bun run dev
    ```

## Project Structure

- `src/` — React components and assets
- `public/` — Static files
